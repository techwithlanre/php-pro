# Doctrine ORM Support

## Annotations and Attributes

PHPDoc highlighting includes common Doctrine annotations and `@ORM\\...`.

PHP 8 attributes highlighting includes `#[ORM\\...]` inside attribute blocks.

Grammar: `syntaxes/php.tmLanguage.json`

## DQL

Standalone `.dql` files are supported via the `dql` language and grammar.

DQL inside `createQuery("...")` strings receives additional semantic highlighting.
